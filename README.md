### Requirements:
* Composer
* Git
* Make
* Node.js
* PHP ^8.0
* SQLite

### Setup:
```sh
$ git clone https://github.com/algins/php-project-lvl3.git
$ cd php-project-lvl3
$ make setup
```

### Run:
```sh
$ make start # Open http://localhost:8000
```
